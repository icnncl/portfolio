# Project Overview
* Project Name: Chantak
* Project Type: Django REST API project management platform
* Development Period: August - September 2025

# Technology Stack 
* Backend Framework: Django 5.2.5 with Django REST Framework 3.16.1
* Database: Postgresql
* Authentication: JWT-based authentication using djangorestframework-simplejwt 5.5.1
* Testing: pytest integration
* Cursor: Development and testing
* Claude Code: Refactoring

# API Endpoints
1. Authentication & JWT - Token management (3       
  endpoints)
2. Users & Auth - Registration, login, profiles,    
   permissions (24+ endpoints)
3. Products - Product management, comparison,       
  search (20+ endpoints)
4. Brands & Manufacturers - Brand and
  manufacturer operations (6 endpoints)
5. Topics & Categories - Category management (6     
  endpoints)
  6. Ingredients - Ingredient management (4
  endpoints)
  7. Tracker - Goals, tasks, records, progress        
  (23+ endpoints)
  8. API Root - Base endpoint

  Key Features:

  - JWT Authentication for secure API access
  - CRUD operations on all resources
  - Custom actions like product comparison,
  progress tracking, favorites
  - GDPR compliance with data export/deletion
  - Multi-user roles (manufacturer, brand owner,      
  retailer)
  - Goal tracking with progress monitoring
  - Email verification & password reset
